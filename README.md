# MERN Auth App 
- Email verification
- Reset Password
- Password Strength Indicator 
- Send Mail with template using `Mail Trap`

### Backend
- express 
- mongoose 
- jsonwebtoken 
- bcryptjs 
- dotenv 
- cors 
- cookie-parser 
- mailtrap

### FrontEnd
- tailwind 
- daisyui
- react-router-dom 
- react-query
- zustand
- framer-motion
